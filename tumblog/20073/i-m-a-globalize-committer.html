I'm a globalize committer

Thanks to the work I did on the alternative storage mechanism for globalize , Josh Harvey invited me on to the Globalize team.

I’m happy to announce that the Globalize team have recently released Globalize For-1.2 which provides full compatibility with Rails 1.2.x and introduces a number of new features some of which I wrote.

In the following series of articles I’m going to be writing about the work I’ve done on Globalize:

  • My first task was to integrate my alternative storage plugin into globalize itself (available in the for-1.2 branch )
  • I then kept myself busy by writing a few tools to help the migration over to this new storage mechanism (partly available in the for-1.2 branch )
  • I then introduced the concept of namespaced view translations (available in the for-1.2 branch )
  • Recently I’ve added a class that encapsulates the idea of an application’s supported locales to globalize trunk.

From this work and work done while developing our latest application I’ve also recently released a plugin that Globalizes Mephisto (http://mephistoblog.com)

P.S. (Shameless plug) If anyone out there would like help globalizing their site/application then we’re available for hire right now
