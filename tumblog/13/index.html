Added tags_with_count to tagtools

Bob released tagtools 0.0.3 not so long ago, and added a very helpful feature, TaggingHelpers.

This allows you to work with tags on a global scale, with all taggable objects.

I wanted to optimise the list of tags you see on the left-hand side of the site, so I added the tags_with_count() method to TaggingHelpers which performs a simple group by query and adds a ‘count’ property to each tag.

Release nightmare

This weekend I did a 32 hour non-stop stint releasing the latest version of CondisLine.com.

This was a massive change as we were changing hardware (2 db servers, 2 web servers, 2 app servers, 2 firewall/proxies) & software (complete revamp of the site under ATG 7.1), communications infrastructure & security, data model migration (If you’ve ever seen the ATG DCS data model you know the kind of hell it was, I had to migrate from our proprietary 5.1.1 version to our proprietary 7.1 version).

At around 4-7 in the morning, I wasn’t particularly lucid but second wind then kicked in and we opened the tap at around 13:00 hrs on the Sunday.

We’re still cleaning up some stuff but considering, I’d say things went really well.
