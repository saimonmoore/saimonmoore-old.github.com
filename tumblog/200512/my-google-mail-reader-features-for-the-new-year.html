My google mail/reader features for the new year

Gmail

  • Option to apply multiple labels to a selection of emails
  • Modify ‘Apply Label’ action to allow application of multiple labels at once (Also for filter action)
  • Add an ‘Apply Label and Archive’ button.
  • Add an out-of-office reply filter action. So I can send out a dedicated email response per contact when I’m unavailable.

Google Reader

  • Make the ‘blogThis’ option a user-definable url
  • If I’m not viewing a post how do I view posts by labels?
