saimonmoore.net now runs multiple globalized sites

I recently upgraded my mephisto_i18n plugin to add complete support for multiple sites.

In the process, I converted the mephisto instance running http://saimonmoore.net into a multisite instance. Currently, the other instance is our companie’s site webtypes.com which I ported over from textpattern.

You can read more about how I did it in the third installment of my globalizing mephisto series.

Other improvements to the site have been:

  • Added better spam protection via Sven’s article
  • Added an email-based contact form so I’ll hopefully not miss any feedback this time.
