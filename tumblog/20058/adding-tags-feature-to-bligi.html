Adding 'tags' feature to bligi

This morning I got up and started to add tags to my Entry object in Bligi using the ‘acts_as_taggable’ mixin from the TagTools project.

Though the rdoc is not up to date the tests available allowed me to get things setup relatively easily.

Only snag was avoiding duplicate tags, which I thought would be handled by the mixin. I’ll just add a validates clause to the Tag object and that should do the trick.
